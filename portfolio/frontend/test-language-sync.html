Language Synchronization Test

This file is for testing the language synchronization between the language store and i18n.

Test Steps:

  1. Open the browser console
  2. Reload the main portfolio page
  3. Check the console logs for language initialization
  4. Change language using the language selector
  5. Verify that projects and experiences are fetched in the new language
  6. Reload the page and verify the selected language persists

Expected Console Output:

  • "Language store initialization - saved language: [lang]"
  • "Language store initialized with: [lang]" or "Language store initialized with default: en"
  • "App mounted - current language: [lang]"
  • "Fetching projects with language: [lang]"
  • "Current i18n locale: [lang]"

When Changing Language:

  • "Changing language to: [lang]"
  • "Setting language to: [lang]"
  • "Language set to: [lang] i18n locale updated"
  • "Data refreshed for language: [lang]"
